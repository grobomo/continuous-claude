Continuous Claude (fork)
========================

Forked from AnandChowdhary/continuous-claude v0.21.0. Upstream removed.

Changes from upstream:
  - Removed update/version-check system (~380 lines)
  - Replaced per-line jq fork loop with single streaming jq process
    (fixes Git Bash fork exhaustion crash on Windows)
  - Added --max-turns flag (default 25) to cap tool-call rounds per
    iteration, preventing runaway iterations
  - Emojis passed as --arg to jq (Windows jq lacks \U escape support)

Files:
  continuous-claude    The script (thin-wrapped in ~/.local/bin/)
  .github/publish.json Grobomo public repo config

Usage:
  continuous-claude -p "prompt" -m 5
  continuous-claude -p "prompt" -m 0 --max-cost 20
  continuous-claude -p "prompt" -m 3 --max-turns 50

Monitor:
  tail -f continuous-claude.log
  gh pr list
  GitHub Mobile push notifications
